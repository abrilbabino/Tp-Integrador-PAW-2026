# TP Maquetado Web 

**Materia:** PAW  
**Fecha de Entrega:** 30/03/2026  

## Introducción
Este repositorio contiene el desarrollo del Trabajo Práctico Nº 1 de PAW, el proyecto se centra en la creación de la página web de una librería ficticia llamada **PAWPrints**.  
En esta entrega se abordan las siguientes tareas:  
- Elaboración del SiteMap para definir la jerarquía de secciones y páginas.  
- Diseño de wireframes low-fi en Figma para las pantallas principales.  
- Maquetado en HTML5 siguiendo las pautas de los wireframes, aplicando correctamente etiquetas semánticas.
